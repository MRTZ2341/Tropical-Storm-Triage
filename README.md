# Tropical-Storm-Triage App
A command line utility system for a hospital operating on incoming patients during a tropical storm like a hurricane.
## Scenario
You are the main leader of the hospital when it comes to recieving patients based on limited resources and try to save people's lives during a hurricane.
## Triage Rules
Patients that arrive are categorized into priority levels and a limited amount of room for them in the queue based on food and water. The lower the level the more sooner they need to be treated. For example:
1 - Critical
2 - Urgent
3 - Moderate
4 - Minor
## Requirments
Java 17 or pretty recent from Java 17.
## Using the App
When executing the file a menu will be printed out with 7 choices in this hospital situation with some patients be already inside the queue after checking resources. Input any number and you will be greeted with the instuctions on what you can do like treat a patient, see the waitlist, add a new patient, or chnage priority.
